
# Packages needed for an Arch installation

## General packages

- xorg-server
- xorg-xinit
- xorg-drivers
- nvidia drivers
- networkmanager
- pulseaudio
- compton / picom
- intel-microcodes

## Programs

- a terminal emulator (kiity, alacritty)
- an image viewer (feh, nomacs)
- a calculator
- firefox
- a file manager (maybe ranger for an in-terminal-file-manager)
- a mail client (geary or thunderbird)
- pdf viewer (zathura)
- color picker
- typora
- vs code
- vlc
- vim, nano
- ufw
- htop
- slock
- fslint (to find duplicated, not really mandatory, but might be useful some day)
- Bitwarden
- Standard Notes
- Office Suite (LibreOffice)
- redshift (config file needs to be created)
